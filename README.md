# Check-Apoint
## A simple bot to check for appointments in the us embassy

__Dependancies__:
- git
- npm
- node

__Clone project__:
```
> git clone https://github.com/yosefgit/check-apoint.git
```

__Install Selenuim driver__:

simpley run provided bash script 
```
> sudo install-selenuim-driver.sh
```

__Install and run__:
```
> npm install
> npm start
```
